How It Works

Up and running in minutes

Four quick steps from install to your first video.

1

Install the Extension

Add GrooveVision to Chrome from the Web Store. No companion app required.

2

Enter Your Last.fm Username

Open the extension popup and type in your Last.fm username. No OAuth, no login — just your username.

3

Add a YouTube API Key

Create a free API key at Google Cloud Console in about two minutes. Paste it into the extension settings — one time, and you're done.

4

Turn On Sync & Play

Flip the sync toggle in the popup, start a song in any music app, and watch the matching YouTube video open automatically.
